# 📝 投稿与贡献指南

Termux-X 及其 Wiki 是一个开放的社区项目，我们非常欢迎每一位用户的贡献！无论您是发现了 Bug、有新的功能建议，还是愿意分享您的使用心得、脚本或美化方案，您的参与都将让 Termux-X 变得更好。

## 🤝 贡献方式

### 1. 完善文档 (Wiki)
如果您发现本文档有错误、遗漏，或者您想补充新的教程：
*   **直接编辑**：在每篇文档页面的底部，点击 **"在 GitHub 上编辑此页"** 链接，即可跳转到 GitHub 对应的源文件进行修改并提交 Pull Request (PR)。
*   **提交 Issue**：如果您不熟悉 Git 操作，也可以在 [GitHub Issues](https://github.com/skilfulwriter/Termux-X/issues) 中反馈您发现的问题或建议。

### 2. 分享脚本与工具
如果您编写了实用的 Shell/Python 脚本，或者移植了新的 Linux 工具：
*   欢迎提交到我们的 **公共仓库**。
*   请在 [GitHub Discussions](https://github.com/skilfulwriter/Termux-X/discussions) 或 [Termux-X 论坛](/online/forum) 发帖分享。
*   优秀的内容将被收录到官方 Wiki 的 [在线脚本](/online/scripts) 或 [下载站](/online/downloads) 中。

### 3. 反馈 Bug与建议
*   如果您在使用 Termux-X App 过程中遇到闪退、报错或功能异常。
*   请前往 [GitHub Issues](https://github.com/skilfulwriter/Termux-X/issues) 提交 Bug 报告。
*   **提交时请注明**：
    *   手机型号与 Android 版本
    *   Termux-X 版本号
    *   复现步骤与报错截图

## 🏆 贡献者致谢
