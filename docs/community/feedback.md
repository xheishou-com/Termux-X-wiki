# 🗣️ 反馈与建议

Termux-X 的发展离不开社区用户的反馈。如果您在使用过程中遇到 Bug、有功能建议，或者想与其他用户交流心得，可以通过以下渠道联系我们。

## 💬 官方交流渠道

### 1. QQ 频道 (推荐)
Termux-X 官方 QQ 频道是获取最新资讯、下载更新和实时交流的主要场所。
*   **点击加入**: [Termux-X 官方频道](https://pd.qq.com/s/g6hj1xy48?b=9)
*   App 内入口：点击侧边栏菜单中的 **"反馈建议"** 按钮即可直接跳转。


![](/img/online/6.png)

### 2. GitHub Issues (Bug 反馈)
如果您发现了程序错误或有明确的功能请求，建议在 GitHub 上提交 Issue，这样可以更方便地追踪问题进度。
*   **提交地址**: [GitHub Issues](https://github.com/xheishou-com/Termux-X-wiki/issues)
*   **提交格式**:
    *   **标题**: 简要描述问题 (如：[Bug] 启动 Kali 时闪退)
    *   **环境**: 手机型号、Android 版本、Termux-X 版本
    *   **复现步骤**: 详细描述如何触发该问题
    *   **截图/日志**: 提供报错截图或日志文件

### 3. 官方论坛
*   **地址**: [Termux-X 社区论坛](https://bbs.x10001.com/forum/90.html)
*   在这里您可以发布长篇教程、分享美化配置或讨论技术话题。



## 📝 反馈技巧
为了让我们更快地帮助您解决问题，反馈时请注意：
*   **保持理性**：描述事实，避免情绪化表达。
*   **提供细节**：不要只说“不能用”，请说明具体的报错信息和操作步骤。
*   **善用搜索**：在提问前，先搜索一下 Wiki 或 Issue 列表，看看是否已经有解决方案。
