# ADB 远程工具

Termux-X 集成了 ADB (Android Debug Bridge) 工具，允许您在移动设备上进行 ADB 调试操作。

## 功能入口

在 Termux-X 工具箱中点击 **ADB远程** 图标。

## 功能说明

点击后将启动内置的 ADB 工具箱 (`AdbToolsActivity`)，通常包含以下功能：
- **ADB 连接管理**：连接到本地或远程设备的 ADB 服务。
- **无线调试**：通过网络进行 ADB 调试。
- **应用管理**：通过 ADB 命令管理已安装的应用。
- **Shell 命令**：执行 ADB Shell 命令。

此功能集成了 `com.davik.adbtools`，为您提供图形化的 ADB 操作体验。
