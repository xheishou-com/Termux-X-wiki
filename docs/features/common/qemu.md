# QEMU 虚拟机

Termux-X 提供了便捷的 QEMU 虚拟机管理功能，支持多种运行模式和系统模拟。

## 功能入口

在 Termux-X 工具箱中点击 **QEMU** 图标即可访问。

## 菜单选项

### 1. 海的QEMU
提供了一键运行 QEMU 脚本的功能。
- **线上脚本**：自动下载并运行最新的 `utqemu.sh` 脚本。
- **本地脚本**：运行本地存储的 `utqemu.sh` 脚本。

### 2. Zero
启动高级 QEMU 管理窗口（RunWindowActivity），提供更丰富的虚拟机配置和管理功能。

### 3. Win7模拟
快速启动 Windows 7 模拟脚本。
- 自动检测并创建 `qemu_win7.sh` 脚本。
- 需要存储权限。

### 4. WinXp
快速启动 Windows XP 模拟脚本。
- 自动检测并创建 `qemu_winxp.sh` 脚本。
- 需要存储权限。
