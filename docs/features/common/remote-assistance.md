# 远程协助

Termux-X 提供了便捷的远程协助功能，便于进行远程控制、屏幕共享和技术支持。该功能基于 MQTT 和 WebSocket 技术实现，支持在复杂的网络环境下（如内网）进行稳定的远程连接。

## 功能入口

在 Termux-X 工具箱中点击 **远程协助** 图标。

## 核心功能

点击后将启动远程协助服务 (`RemoteControlService`)，主要功能包括：

### 1. 屏幕共享
*   **实时投屏**：将手机屏幕画面实时传输给远程协助者，方便对方查看当前操作界面和报错信息。
*   **画质调节**：根据网络状况自动或手动调节传输画质，确保流畅性。

### 2. 远程控制
*   **手势操作**：支持远程点击、滑动等手势操作，协助者可以直接控制您的设备进行配置。
*   **文本输入**：支持远程输入文本，方便输入复杂的命令或代码。

### 3. 连接管理
*   **ID 连接**：每个设备拥有唯一的协助 ID，通过 ID 即可建立连接，无需复杂的 IP 配置。
*   **断开控制**：用户随时可以主动断开连接，保障隐私安全。
*   **后台服务**：支持后台运行，即使切换到其他应用也能保持协助连接。

## 使用场景
*   **技术支持**：当您遇到 Termux 配置问题无法解决时，可以邀请大佬远程协助排查。
*   **远程教学**：通过屏幕共享演示操作步骤，进行实时的教学演示。
*   **设备管理**：远程管理自己的备用机或服务器设备。

> **注意**：使用远程控制功能需要授予 Termux-X **辅助功能 (Accessibility)** 和 **屏幕录制** 权限。
