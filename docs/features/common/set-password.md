# 设置密码 (Set Password)

在使用 SSH 远程连接或 VNC 服务时，您需要设置一个用户密码。

## 功能说明
该功能会调用 Linux 标准的 `passwd` 命令来修改当前用户的登录密码。

## 操作步骤
1. 点击 "设置密码" 按钮。
2. 终端会提示 `New password:`。
3. **输入您的密码**（注意：输入时屏幕上**不会显示**任何字符，这是正常的 Linux 安全机制）。
4. 按回车键。
5. 终端提示 `Retype new password:`。
6. **再次输入相同的密码**并按回车。
7. 如果两次输入一致，将提示 `passwd: password updated successfully`。

## 手动执行命令
```bash
passwd
```

## 忘记密码？
如果您忘记了密码，可以直接再次点击 "设置密码" 或输入 `passwd` 命令重新设置即可（在 Termux 环境下，当前用户通常有权限直接修改自己的密码）。
