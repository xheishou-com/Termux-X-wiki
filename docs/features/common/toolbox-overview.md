# Termux-X 工具箱

Termux-X 工具箱集合了多种实用的系统管理和渗透测试辅助工具，旨在简化复杂操作，提升工作效率。

## 功能列表

### 1. [ADB远程](/features/common/adb-remote)
通过网络进行 ADB 连接和调试，无需 USB 线缆，方便进行无线调试。

### 2. [容器切换](/features/common/container-switching)
快速在不同的 Linux 容器环境之间进行切换，管理多个独立的开发或测试环境。

### 3. [备份/恢复](/features/common/backup-restore)
一键备份和恢复您的 Termux 环境数据，防止数据丢失，方便迁移设备。

### 4. [QEMU](/features/common/qemu)
利用 QEMU 模拟器运行 x86_64、arm 等架构的操作系统，扩展 Termux 的运行能力。

### 5. [SSH连接](/features/common/ssh-connect)
内置便捷的 SSH 客户端功能，快速连接远程服务器，支持密钥管理。

### 6. [Webshell](/features/common/webshell)
提供基于 Web 的终端访问界面，通过浏览器即可操作 Termux。

### 7. [Cobalt Strike](/features/common/cobalt-strike)
集成了 Cobalt Strike 相关工具或辅助功能，用于红队渗透测试协作。

### 8. [定时任务](/features/common/scheduled-tasks)
管理和设置定时任务（Cron jobs），自动化执行脚本或维护工作。

### 9. [远程协助](/features/common/remote-assistance)
提供远程协助功能，允许他人通过网络连接到您的终端进行协助或演示。
