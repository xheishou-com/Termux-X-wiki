# 常用 Kali 工具

Termux-X 集成了 Kali NetHunter 的强大工具集，并提供了便捷的分类菜单，让您可以快速查找和使用各种渗透测试工具。

## 功能入口

在 Termux-X 工具箱或菜单中找到 **常用kali工具** (或类似名称) 选项。

## 工具分类

点击后将显示工具分类菜单，包含但不限于以下类别：

- **信息收集**：Nmap, TheHarvester, Recon-ng 等
- **漏洞扫描**：Nikto, OpenVAS 等
- **Web应用**：Sqlmap, Burp Suite, Zaproxy 等
- **密码破解**：John, Hashcat, Hydra 等
- **无线安全**：Aircrack-ng, Kismet 等
- **漏洞利用**：Metasploit, SearchSploit 等
- **后渗透**：Empire, Mimikatz 等
- **社会工程**：SET, Phishing Framework 等

## 使用说明

1. 选择相应的分类。
2. 在弹出的列表中点击要使用的工具。
3. Termux-X 将自动在 Kali 环境中启动该工具。
