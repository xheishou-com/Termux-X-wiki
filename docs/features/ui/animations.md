# ✨ 背景动画 (雪花与粒子)

Termux-X 支持在终端背景上显示动态效果，增添视觉趣味。目前支持 **雪花** 和 **粒子** 两种特效。

## ❄️ 雪花特效 (Snowflakes)

开启后，终端背景会有雪花缓缓飘落，营造出冬日氛围。

### 使用方法
1.  打开侧边栏菜单。
2.  找到 **雪花** 按钮。
3.  点击即可开启或关闭。
    *   按钮显示“雪花开”表示当前已开启。
    *   按钮显示“雪花关”表示当前已关闭。

## 🌌 粒子特效 (Particles)

粒子特效会显示漂浮的粒子，并在粒子之间建立连线，类似科技感的网络结构。

### 使用方法
1.  打开侧边栏菜单。
2.  找到 **粒子动画** 按钮。
3.  点击即可开启或关闭。

## ⚠️ 性能提示

*   开启背景动画会消耗一定的 CPU 和 GPU 资源，可能会增加耗电量。
*   在低性能设备上，可能会导致终端响应变慢。如果遇到卡顿，建议关闭动画效果。
