# ✨ 背景动画

Termux-X 支持在终端背景上叠加动态特效，增加视觉趣味。目前主要支持 **雪花** 和 **粒子** 两种效果。

## ❄️ 雪花特效 (Snowflakes)

模拟冬日雪花飘落的效果，营造唯美氛围。

### 使用方法
1.  打开侧边栏菜单。
2.  找到 **雪花** (Snowflake) 按钮。
3.  点击即可切换状态：
    *   **雪花开**：开启特效。
    *   **雪花关**：关闭特效。

## 🌌 粒子特效 (Particles/Rain)

显示科技感的浮动粒子或类似黑客帝国的数字雨效果（视具体版本实现而定，通常称为粒子或 Rain）。

### 使用方法
1.  打开侧边栏菜单。
2.  找到 **粒子动画** (Particle/Rain) 按钮。
3.  点击即可开启或关闭。

## ⚠️ 性能提示
*   开启背景动画会占用一定的 GPU 渲染资源。
*   在低端设备上，如果感觉终端输入有延迟，建议关闭背景动画。
