# 🎨 美化设置

**美化设置** 允许您自定义 Termux-X 的外观，包括颜色主题、背景透明度、光标样式等，打造个性化的终端体验。

## 🚀 功能说明

通过侧边栏的 **美化设置** 按钮，您可以访问以下配置：

### 1. 颜色主题 (Color Scheme)
*   提供多种预设配色方案（如 Solarized, Monokai, Dracula 等）。
*   自定义前景色（文本颜色）和背景色。

### 2. 背景设置
*   **图片背景**：选择一张图片作为终端背景。
*   **透明度**：调整背景图片的透明度或纯色背景的透明度。
*   **模糊度**：为背景添加模糊效果，提高文字可读性。

### 3. 光标样式
*   调整光标的形状（块状、下划线、竖线）。
*   设置光标是否闪烁。

## 📝 使用方法

1.  打开侧边栏菜单。
2.  点击 **美化设置**。
3.  在弹出的对话框或界面中调整各项参数。
4.  更改即时生效。
