# 🎨 美化设置

**美化设置** 允许您自定义 Termux-X 的外观，包括颜色主题、背景透明度、光标样式等，打造个性化的终端体验。

## 🚀 功能说明

通过侧边栏的 **美化设置** (Beautify) 按钮，您可以打开配置对话框，包含以下选项：

### 1. 字体颜色 (Font Color)
*   **颜色条**：拖动滑块选择终端文字的颜色。
*   **透明度**：调整文字的透明度 (Alpha)。

### 2. 背景颜色 (Background Color)
*   **颜色条**：拖动滑块选择终端背景的颜色。
*   **注意**：如果您设置了背景图片，背景颜色将作为叠加层显示（取决于透明度设置）。

### 3. 背景图片/视频 (Background Image/Video)
*   **点击预览区域**：点击对话框上方的图片预览区域，可以打开文件选择器。
*   **选择文件**：支持选择图片（JPG/PNG）或视频文件作为背景。
*   **自动适配**：选择后会自动应用并保存。

### 4. 界面调整
*   **背景透明度**：调整背景图片/视频的显示透明度。
*   **文字阴影/显示**：通过开关控制是否增强文字显示效果（如添加阴影以提高在复杂背景上的可读性）。

## 📝 使用方法

1.  打开侧边栏菜单。
2.  找到并点击 **美化设置** 按钮。
3.  在弹出的对话框中进行调整。
4.  点击对话框外部或返回键关闭，设置即时生效。
