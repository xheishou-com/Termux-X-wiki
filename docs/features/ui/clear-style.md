# 🧹 清空美化

**清空美化** (Clear Style) 功能用于一键重置 Termux-x 的所有外观设置，恢复到初始默认状态。

## 🚀 功能说明

执行此操作将清除以下所有自定义配置：
*   背景图片和视频背景。
*   自定义的字体颜色和背景颜色。
*   背景动画效果（雪花、粒子等）。
*   字体大小和类型设置（恢复默认字体）。
*   其他界面微调参数。

## 📝 使用方法

1.  打开侧边栏菜单。
2.  找到 **清空美化** 按钮（通常在美化功能区域）。
3.  点击按钮。
4.  App 将会自动刷新或重启 Activity 以应用默认设置。

## ⚠️ 警告
*   **操作不可撤销**：一旦清空，您之前的配色方案和背景设置将无法自动恢复。
*   **建议**：如果您配置了复杂的配色方案，建议先截图保存参数。
