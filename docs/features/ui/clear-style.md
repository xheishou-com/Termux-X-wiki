# 🧹 清空美化

**清空美化** 功能用于将 Termux-X 的外观设置重置为默认状态。

## 🚀 功能说明

执行此操作将：
*   清除自定义的背景图片或视频。
*   关闭背景动画（雪花、粒子）。
*   重置颜色主题为默认值。
*   重置字体设置。
*   重置其他界面调整。

## 📝 使用方法

1.  打开侧边栏菜单。
2.  点击 **清空美化** 按钮。
3.  App 可能会重启或刷新界面以应用默认设置。

## ⚠️ 警告

*   **此操作不可逆**：一旦执行，您之前的个性化配置将会丢失。
*   **建议**：在执行前，如果您有特别满意的配置，建议先截图或记录参数。
