# 🔡 字体设置

**字体设置** 允许您更换终端显示的字体，以适应不同的编程需求和审美偏好。

## 🚀 功能说明

Termux-X 内置了字体管理功能，支持一键切换常用编程字体。

### 使用方法
1.  打开侧边栏菜单。
2.  点击 **字体设置** (Font) 按钮。
3.  将打开字体选择列表或管理器（FontActivity）。
4.  浏览并选择您喜欢的字体。
5.  应用后，终端字体将立即更新。

## 💡 自定义字体
如果内置列表不能满足需求，您通常可以通过以下方式手动替换字体：
1.  准备一个 `.ttf` 格式的字体文件。
2.  将其重命名为 `font.ttf`。
3.  放入 `~/.termux/` 目录下（如果该目录不存在需手动创建）。
4.  重启 App 或执行 `termux-reload-settings`。
*(注：使用侧边栏的字体设置功能通常更方便，它会自动处理文件替换)*
