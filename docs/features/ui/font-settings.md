# 🔡 字体设置

**字体设置** 允许您更改终端显示的字体样式和大小，以获得最佳的阅读体验或编程风格。

## 🚀 功能说明

Termux-X 内置了多种流行的编程字体，同时也支持导入自定义字体。

### 支持的字体类型
*   **等宽字体 (Monospace)**：推荐用于编程和终端显示，确保字符对齐。
*   **Powerline 字体**：支持特殊符号显示（如 Oh My Zsh 主题所需的箭头和图标）。

## 📝 使用方法

1.  打开侧边栏菜单。
2.  点击 **字体设置**。
3.  在列表中选择您喜欢的字体。
4.  终端字体将立即更新。

## 💡 自定义字体
如果您想使用自己的字体文件（`.ttf` 或 `.otf`）：
1.  将字体文件放入 Termux-X 的特定目录（通常是 `~/.termux/font.ttf`，具体视版本而定，建议优先使用内置选择器）。
2.  或者在字体设置界面寻找“导入”或“自定义”选项。
