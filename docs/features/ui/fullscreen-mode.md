# 📺 全屏模式

**全屏模式** 可以隐藏 Android 系统的状态栏和导航栏，为您提供更沉浸式的终端体验和更大的显示空间。

## 🚀 功能说明

*   **沉浸式体验**：隐藏顶部状态栏（时间、电量等）和底部导航键。
*   **最大化视野**：在小屏幕设备上特别有用，能显示更多行代码。

## 📝 使用方法

1.  打开侧边栏菜单。
2.  点击 **全屏模式** 按钮。
    *   **开启**：状态栏和导航栏消失。
    *   **关闭**：再次点击即可恢复系统栏显示。

## 💡 提示

*   在全屏模式下，您通常可以通过从屏幕边缘向内滑动来临时唤出系统导航栏或状态栏。
*   结合 **横屏模式** 使用，效果更佳。
