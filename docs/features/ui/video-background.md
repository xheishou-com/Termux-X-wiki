# 🎥 视频背景

**视频背景** 允许您将本地视频文件设置为终端的动态背景，打造极致的个性化界面。

## 🚀 功能说明

*   支持常见的视频格式（如 MP4）。
*   视频将循环播放。
*   视频上方会覆盖一层半透明的终端界面，确保文字依然可见。

## 📝 使用方法

1.  打开侧边栏菜单。
2.  点击 **视频背景** (或底部工具栏中的视频图标)。
3.  在弹出的文件选择器中，选择您设备上的视频文件。
4.  确认后，视频将作为背景开始播放。

## ⚠️ 注意事项

*   **性能消耗**：视频背景比静态图片或简单动画消耗更多的电量和系统资源。
*   **可读性**：建议选择画面较暗或对比度较低的视频，以免干扰终端文字的阅读。
*   **音频**：如果视频包含音频，可能会同时播放声音（取决于具体实现），建议使用静音视频。

## 🔄 清除背景
如果您想恢复默认背景，可以使用侧边栏中的 **[清空美化](/features/ui/clear-style)** 功能。
