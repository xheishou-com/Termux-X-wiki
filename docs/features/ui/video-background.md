# 🎥 视频背景

**视频背景** 是 Termux-X 的一项特色功能，允许将 MP4 等视频文件设置为终端的动态壁纸。

## 🚀 设置方法

视频背景的设置入口整合在 **美化设置** 中：

1.  打开侧边栏菜单。
2.  点击 **美化设置** (Beautify) 按钮。
3.  在弹出的对话框中，**点击顶部的图片预览区域**。
4.  系统将打开文件选择器（或 ImageActivity）。
5.  在文件选择器中找到并选择您的视频文件。
6.  确认后，视频将自动作为背景循环播放。

## ⚠️ 注意事项

*   **互斥性**：视频背景与图片背景互斥，设置新的视频会自动清除之前的图片背景，反之亦然。
*   **性能影响**：高清视频背景可能会增加耗电量和 CPU/GPU 占用，建议在电量充足时使用。
*   **可读性**：建议配合 **美化设置** 中的“背景颜色透明度”或“文字增强”功能，以确保终端文字清晰可见。

## 🔄 清除视频
如果您想移除视频背景，可以使用侧边栏的 **[清空美化](/features/ui/clear-style)** 功能，或者在美化设置中重新选择一张纯色背景/图片。
