# ✍️ 修改欢迎语 (MOTD)

**欢迎语** (Message of the Day, MOTD) 是您每次打开 Termux-X 新会话时在顶部显示的问候信息。默认情况下，它会显示一些帮助提示。您可以将其修改为任何您喜欢的文本或 ASCII 艺术。

## 🚀 如何修改

1.  打开 Termux-X 侧边栏菜单。
2.  找到 **美化功能** 区域。
3.  点击 **修改欢迎语** 按钮。
4.  在弹出的编辑器中输入您想要显示的内容。
5.  保存并退出编辑器。
6.  新建一个会话或重启 App，即可看到新的欢迎语。

## 💡 提示

*   支持标准文本和部分 ASCII 字符。
*   如果您想恢复默认欢迎语，只需清空内容或删除 `motd` 文件（位于 `usr/etc/motd`，如果存在）。
