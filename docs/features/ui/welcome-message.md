# ✍️ 修改欢迎语

**欢迎语** (Start Msg / MOTD) 是您每次打开新终端会话时显示的问候文本。Termux-X 允许您轻松自定义这段内容。

## 🚀 修改方法

1.  打开侧边栏菜单。
2.  找到 **修改欢迎语** (Start Msg) 按钮。
3.  点击后将打开一个简单的文本编辑器。
4.  在编辑器中输入您喜欢的文字、ASCII 艺术画或提示信息。
5.  点击保存并退出。
6.  **新建会话**：下次打开新的终端窗口时，您将看到修改后的欢迎语。

## 💡 进阶技巧
*   支持 Linux 标准的 ASCII 颜色代码（如果编辑器支持预览则更佳，否则需手动输入转义序列）。
*   您可以利用此功能显示待办事项、励志名言或服务器状态提示。
