# 停止桌面 (Stop Desktop)

当您完成图形化界面的操作，或者遇到桌面卡死、显示异常时，可以使用 "停止桌面" 功能。

## 功能说明
该功能用于强制终止 X11 相关的后台进程，释放系统资源。

## 执行流程
系统会查找所有包含 `com.termux.x11` 名称的进程并将其终止。

## 手动执行命令
```bash
pkill -f com.termux.x11
```

## 常见问题
- **无法停止？** 如果点击按钮后桌面依然存在，请尝试在终端中多执行几次上述命令，或直接在安卓任务管理中划掉 Termux:X11 应用。
- **重新启动**：停止后如果需要重新启动，请等待几秒钟确保进程完全退出后再点击 "启动桌面"。
