# 🐳 Docker 检查 (Docker Check)

**Docker 检查** 功能用于检测您的 Android 设备内核是否支持运行 Docker 容器。

## ⚠️ 前置要求

*   **Root 权限**：此功能需要 Root 权限来执行检测脚本和读取内核配置。

## 🚀 功能说明

由于 Android 设备的内核碎片化严重，并非所有设备都能直接运行 Docker。此功能通过运行官方或社区提供的检测脚本 (`check-config.sh`)，分析当前内核配置。

### 使用方法

1.  打开侧边栏菜单。
2.  找到并点击 **Docker检查** 按钮。
3.  应用将请求 Root 权限。
4.  脚本将自动下载（或使用内置版本）并执行。
5.  **查看结果**：终端窗口将输出详细的检测报告。

### 结果解读

报告通常包含以下部分：
*   **Generally Necessary**: 必须开启的内核选项。如果这里有 `missing`，则 Docker 很可能无法启动。
*   **Optional Features**: 可选功能。缺失可能会限制某些高级网络或存储功能。

如果您的设备缺少关键内核特性，通常需要自行编译并刷入支持 Docker 的定制内核。
