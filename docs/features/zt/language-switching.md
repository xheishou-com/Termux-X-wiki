# 🌐 语言切换 (Language Switching)

**语言切换** 功能允许您在 Termux-X 的界面语言之间进行切换，以适应您的使用习惯。

## 🚀 支持的语言

目前 Termux-X 主要支持以下语言：

*   **中文 (Chinese)**：简体中文界面。
*   **English**：英文界面。

## 📝 使用方法

1.  打开侧边栏菜单。
2.  点击 **语言切换** 按钮（图标通常为文/A）。
3.  在弹出的菜单中选择您想要的语言：
    *   点击 **中文** 切换到中文界面。
    *   点击 **English** 切换到英文界面。
4.  **自动生效**：应用界面会立即尝试刷新以显示新语言，部分界面可能需要重启应用才能完全更新。

## ⚠️ 注意事项

*   此设置主要影响 Termux-X 应用本身的 UI（如菜单、设置、提示框）。
*   它**不会**更改 Linux 终端环境内部的 `LOCALE` 或语言设置（如 `LANG` 环境变量）。如果您需要更改终端内的语言环境，请使用 Linux 命令（如 `export LANG=zh_CN.UTF-8`）进行配置。
