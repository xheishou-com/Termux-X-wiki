# 📂 打开目录 (Open Directory)

**打开目录** 功能允许您通过外部文件管理器快速访问和管理 Termux-X 的内部文件系统。这比在命令行中使用 `ls` 和 `cd` 命令更加直观和便捷。

## 🚀 功能说明

*   **直观管理**：使用图形化界面查看、复制、移动、删除和重命名文件。
*   **快速访问**：直接跳转到 Termux-X 的 `home` 目录或其他指定位置。
*   **文件交互**：方便地在 Termux-X 内部存储和手机 SD 卡之间传输文件。

## 📦 插件要求

此功能依赖于 **Termux-X 专用目录插件** (ZeroTermux Files Plugin)。
*   **自动检测**：点击按钮时，应用会自动检测是否已安装该插件。
*   **自动提示**：如果未安装，应用会弹出提示框，引导您前往下载并安装。
    *   下载地址通常为：`https://xheishou.com/update/apk/files.apk`

## 📝 使用方法

1.  打开侧边栏菜单。
2.  点击 **打开目录** 按钮。
3.  **如果已安装插件**：会自动跳转到文件管理器界面，显示 Termux 文件。
4.  **如果未安装插件**：
    *   会弹出提示框：“没有安装Termux-X专用目录插件 是否前往下载”。
    *   点击 **是 (Yes)** 跳转到下载页面。
    *   下载并安装 APK 文件。
    *   安装完成后，再次点击按钮即可使用。
