# ⚙️ Termux-X 设置 (ZT Settings)

**Termux-X 设置** (ZT Settings) 是 Termux--X 专有的设置面板，提供了一系列增强功能和个性化选项，独立于 Termux 原生设置。

## 🚀 功能选项

点击侧边栏的 **ZT设置** 按钮即可进入此菜单，包含以下主要功能：

### 1. ZT 下载服务器 (ZT Download Service)
*   **功能**：开启一个本地 HTTP 服务器，允许局域网内的其他设备访问和下载 Termux-X 中的文件。
*   **默认端口**：`19956`
*   **⚠️ 注意**：此功能默认**无账号密码保护**，开启后请注意个人隐私和文件安全！

### 2. 界面与交互 (Interface & Interaction)
*   **侧边栏自动关闭**：开启后，当调出软键盘输入时，侧边栏（左侧或右侧）会自动收起，避免遮挡视线。
*   **点击“美化设置”关闭左侧栏**：设置点击美化设置按钮时是否同时关闭侧边栏。
*   **禁止工具箱显示**：如果双指缩放字体时与工具箱手势发生冲突，可在此临时关闭工具箱显示。
*   **还原音量键**：禁用 Termux-X 默认的“音量键呼出侧边栏”功能，恢复音量键的系统默认行为（调节音量）。

### 3. 键盘与输入 (Keyboard)
*   **打开键盘规则模式**：强制使用 Termux-X 的键盘规则配置。
*   **设置键盘规则**：允许用户自定义输入法的行为和规则，提高输入效率。

### 4. 日志与调试 (Logs & Debugging)
*   **输出 LOG 到存储卡**：将应用的运行日志输出到 SD 卡目录 `xinhao/ZeroTermuxLog/ZeroTermux.log`。
    *   **⚠️ 注意**：开启此功能可能会影响应用性能，仅建议在排查问题时开启。

### 5. 其他 (Others)
*   **关于 Termux-X**：查看版本信息、开源协议和鸣谢列表。
*   **还原/扩展官方版本**：安装或修复 Termux 官方组件（如 Termux:API, Termux:Float 等）。

## 📝 使用方法

1.  打开侧边栏菜单。
2.  点击 **ZT设置** 按钮。
3.  在列表中点击相应的开关或选项进行配置。
4.  部分设置可能需要重启应用才能生效。
