# 浏览器终端 (Browser Terminal)

**功能简介**：
启动一个本地 Web 服务（通常基于 ttyd 或 gotty），将 Termux 终端界面映射到网页上。这允许您在同一局域网下的电脑、平板或其他手机的浏览器中访问和控制当前的 Termux 会话。

**包含功能**：
- **启动浏览器终端**：开启服务，屏幕上会显示访问地址（如 `http://192.168.x.x:xxxx`）。
- **停止浏览器终端**：关闭服务，释放端口。

**使用方法**：
1. **启动**：
   - 打开工具箱，点击 **启动浏览器终端**。
   - 记下弹出的 URL 地址。
   - 在电脑浏览器中输入该地址，即可看到 Termux 界面并进行输入操作。
2. **停止**：
   - 使用完毕后，打开工具箱，点击 **停止浏览器终端**。

**注意**：
请确保设备连接了 Wi-Fi，且访问设备处于同一局域网内。
