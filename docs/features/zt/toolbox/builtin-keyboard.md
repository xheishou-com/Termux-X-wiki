# 内置键盘 (Built-in Keyboard)

**功能简介**：
Termux-X 提供了一个**自定义的全功能虚拟键盘**（ZeroTermux Custom Keyboard），旨在替代系统的软键盘。

**功能特点**：
- **全键位布局**：提供完整的 PC 风格键盘布局，包含独立的 `ESC`、`CTRL`、`ALT`、`TAB`、`FN` 以及 `F1-F12` 功能键。
- **终端优化**：专为终端操作优化，解决了部分第三方输入法无法正确输入特殊字符或控制码的问题。
- **沉浸体验**：启用时会隐藏系统软键盘和 Termux 原生的单行扩展键栏（Extra Keys），提供更统一的输入体验。

**使用方法**：
1. 双击终端屏幕打开工具箱。
2. 点击 **内置键盘** 按钮。
   - **开启**：如果当前使用的是系统输入法，点击后将切换为 Termux-X 内置键盘。
   - **关闭**：如果当前正在使用内置键盘，点击后将关闭它，并恢复系统输入法和原生扩展键栏。

**适用场景**：
- 需要大量使用功能键（如 Vim/Emacs 操作）时。
- 系统输入法占用屏幕空间过大或干扰输入时。
- 在没有外接物理键盘的情况下进行复杂的代码编辑。
