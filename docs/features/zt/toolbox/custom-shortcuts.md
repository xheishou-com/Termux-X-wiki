# 自定快捷方式 (Custom Shortcuts)

**功能简介**：
除了预置的常用目录外，此功能允许您创建指向 `/sdcard` 下任意目录的自定义软链接。这对于快速访问特定的工作目录、项目文件夹或媒体库非常有用。

**工作原理**：
该功能会在 `~/xinhaoLink/` 目录下创建一个与您输入目录同名的软链接，指向 `/sdcard/` 下的对应路径。

**使用方法**：
1. 双击终端屏幕打开工具箱。
2. 点击 **自定快捷方式**。
3. **输入路径**：
   - 在弹出的对话框中，输入 `/sdcard/` 下的目标文件夹名称（相对路径）。
   - **示例**：如果您想链接 `/sdcard/Movies`，只需输入 `Movies`。
   - **默认值**：如果留空直接点击确定，默认会链接 `/sdcard/xinhao/data` 到 `~/xinhaoLink/xinhao_data`。
4. **确认**：点击确定后，系统会创建软链接。
5. **访问**：在终端中输入 `cd ~/xinhaoLink/您的文件夹名` 即可进入。

**路径处理规则**：
- 如果输入的路径包含斜杠 `/`，在创建链接名称时会被替换为下划线 `_`（例如输入 `DCIM/Camera`，链接名可能变为 `DCIM_Camera`）。
- 确保您输入的目录在 SD 卡中实际存在，否则虽然链接创建成功，但无法进入。
