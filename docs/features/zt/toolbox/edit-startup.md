# 编辑启动文件 (Edit Startup File)

**功能简介**：
提供了一个内置的简易文本编辑器，直接打开当前 Shell 的启动配置文件（如 `.bashrc`）。

**功能特点**：
- 无需记忆 vim 或 nano 命令，图形化界面编辑。
- 支持保存并立即应用（部分配置需重启会话生效）。

**常见用途**：
- 添加环境变量（`export PATH=$PATH:...`）。
- 设置命令别名（`alias ll='ls -l'`）。
- 自定义启动欢迎语（`echo "Welcome back!"`）。

**使用方法**：
1. 双击终端屏幕打开工具箱。
2. 点击 **编辑启动文件** 按钮。
3. 在弹出的编辑框中修改脚本内容。
4. 点击保存按钮。
