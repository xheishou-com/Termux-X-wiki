# 网络 Linux 目录 (Network Linux Directory)

**功能简介**：
基于 FileBrowser 项目，提供一个功能强大的 Web 文件管理器。通过它，您可以在电脑浏览器上可视化地管理 Termux 中的文件（上传、下载、重命名、编辑代码、播放媒体等）。

**包含功能**：
- **安装网络linux目录**：首次使用需点击安装，下载 FileBrowser 二进制文件。
- **启动网络linux目录**：开启 FileBrowser 服务，默认端口通常为 19952 或随机端口。
- **停止网络linux目录**：关闭服务。

![](/img/zt/3.png)

**使用方法**：
1. **安装**（仅首次）：点击 **安装网络linux目录**，等待下载完成。
2. **启动**：点击 **启动网络linux目录**。
3. **访问**：在浏览器中输入提示的地址（如 `http://IP:19952`）。
   - 默认用户名/密码通常为 `admin/xheishou.com`（具体请看启动时的提示信息）。
4. **管理文件**：在网页端进行文件操作。
5. **停止**：点击 **停止网络linux目录** 关闭服务。
