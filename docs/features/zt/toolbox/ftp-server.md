# FTP 服务 (FTP Server)

**功能简介**：
Termux-X 内置了一个 FTP 服务器管理功能，允许您通过 FTP 协议在电脑或其他设备上管理手机（Termux 环境）中的文件。

**功能特点**：
- **图形化管理**：通过弹出的对话框（FtpWindowsDialog）管理服务状态。
- **自定义端口**：默认端口通常为 2121 或 8021（具体视配置而定，代码中也提及 19956 用于其他服务，FTP 具体端口需在启动时查看）。
- **匿名/用户模式**：支持匿名访问或设置用户名密码。

**使用方法**：
1. 双击终端屏幕打开工具箱。
2. 点击 **FTP服务**。
3. 系统会打开一个 FTP 管理窗口（悬浮窗或对话框）。
4. 点击 **启动** (Start) 按钮。
5. 窗口将显示当前的 FTP 地址（例如 `ftp://192.168.1.5:2121`）。
![](/img/zt/5.jpg)

6. 使用文件管理器（如 ES文件浏览器、MT管理器）或电脑上的 FTP 客户端连接该地址。示例：ftp://ftp:ftp@192.168.10.11:2323/

其中 ftp:ftp 为默认用户名密码，您可以在启动时自定义。
![](/img/zt/5.png)
**注意事项**：
- 确保您的手机和电脑连接在同一个 Wi-Fi 网络下。
- 如果连接失败，请检查手机的防火墙设置或尝试更改端口。
- `R.string.ftp_msg` 提示核心基于 Apache 改进，修复了连接异常问题。
