# 创建数据包信息 (Create Package Info)

**功能简介**：
此功能允许用户为当前的 Termux 容器/环境编写一个自定义的**备注信息（Data Message）**。这个信息会显示在 Termux-X 的主菜单（侧边栏）最右侧，帮助您快速识别当前容器的用途或配置状态。

**适用场景**：
- 当您使用 Termux-X 的多容器功能（系统切换）时，可以为每个容器标注名称（如 "Python开发环境"、"Web服务器"、"渗透测试"）。
- 记录当前环境的重要配置信息或提醒。

**使用方法**：
1. 双击终端屏幕打开工具箱。
2. 点击 **创建数据包信息**。
3. 在弹出的编辑框中，输入您想要显示的文字内容（支持多行）。
4. 点击 **保存**。
5. **查看效果**：向右滑动打开侧边栏，您设置的信息将显示在侧边栏的顶部或特定区域（取决于当前 UI 布局）。

**存储位置**：
信息被保存在 `~/ZtInfo/data.config` 文件中。您也可以直接编辑此文件来修改信息。
