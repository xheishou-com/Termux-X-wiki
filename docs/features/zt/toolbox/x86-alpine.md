# X86 Alpine

**功能简介**：
利用 QEMU 模拟器在 Termux 中运行 X86 架构的 Alpine Linux 系统。

**为什么需要它？**
- 某些闭源软件或旧版工具只提供 X86/Linux 的二进制文件，不支持 ARM 架构。
- 需要进行跨架构的编译或测试。
- Alpine Linux 极其轻量，启动速度快，适合作为临时的 X86 容器。

**使用方法**：
1. 双击终端屏幕打开工具箱。
2. 点击 **X86 Alpine**。
3. 如果是首次运行，可能会自动下载镜像文件。
4. 启动后，您将进入一个模拟的 X86 Shell 环境。
5. 输入 `exit` 退出模拟环境。

**性能注意**：
由于涉及 CPU 架构模拟（ARM 转 X86），运行速度会显著低于原生环境，不建议运行重负载应用。
