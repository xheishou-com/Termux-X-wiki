# 🖥️ VNC 远程桌面

VNC (Virtual Network Computing) 功能允许您通过图形界面远程控制 Termux-X 中的 Linux 环境。这对于运行需要图形界面的应用程序（如桌面环境、GUI 工具）非常有用。

## 🚀 功能模式

点击侧边栏的 **VNC** 按钮后，您将看到以下三个选项：

### 1. 快速 VNC (Quick VNC)
*   **用途**：最快捷的启动方式，使用默认配置。
*   **适用场景**：日常快速查看或简单操作。
*   **默认端口**：通常为 `5901`。

### 2. 自定 VNC (Custom VNC)
*   **用途**：允许您自定义连接参数。
*   **配置项**：
    *   **IP 地址**：指定监听的 IP 地址（如 `127.0.0.1` 或 `0.0.0.0`）。
    *   **端口**：自定义 VNC 服务端口。
    *   **密码**：设置连接密码以增强安全性。

### 3. 高级 VNC (Advanced VNC)
*   **用途**：提供更多高级启动参数。
*   **功能**：可以调整分辨率、颜色深度等高级选项，适合对性能或显示效果有特殊要求的用户。

## 📦 插件要求

使用 VNC 功能需要安装 **VNC 插件**。
*   如果您尚未安装，点击 VNC 按钮时，应用会提示您前往 **下载站** 下载并安装相关插件。
*   请确保给予插件必要的运行权限。

## 📝 使用步骤

1.  打开侧边栏菜单。
2.  点击 **VNC** 按钮。
3.  在弹出的菜单中选择您需要的模式（快速、自定或高级）。
4.  等待服务启动。
5.  启动成功后，您可以使用 VNC 客户端（或应用内置的查看器）连接到指定的 IP 和端口进行操作。
