# 🤖 AI 对话 (AI Chat)

Termux-X 的 AI 对话是一个功能强大的图形化聊天界面，它不仅仅是一个问答机器人，更是一个能够与您的系统深度交互的智能助手。

## 🌟 核心功能

1.  **多轮对话记忆**：自动保存您的对话历史，随时回溯之前的讨论。
2.  **MCP 工具集成**：通过 Model Context Protocol (MCP) 协议，AI 可以直接读取文件、执行命令、管理系统。
3.  **Markdown 渲染**：完美支持代码高亮、表格、列表等 Markdown 格式。
4.  **智能上下文**：自动携带系统提示词（System Prompt），专注于 Linux/Termux 问题解决。

## 🚀 使用指南

### 1. 进入 AI 对话
在 Termux-X 侧边栏中，点击 **"AI对话"** 图标（通常为气泡图标）。

![](/img/AI/25.png)

### 2. 会话管理
*   **新建对话**：点击右上角的 "+" 按钮开启新话题。
*   **历史记录**：点击左上角的菜单按钮（或向右滑动）打开侧边栏，查看和切换历史会话。
*   **删除对话**：在历史列表中长按某条记录即可删除。

### 3. MCP 工具使用 (进阶)
如果您在 [AI 配置](/guide/ai-configuration) 中启用了 MCP 服务器（如 `ssh-mcp` 或 `filesystem-mcp`），AI 将获得“手脚”。

**示例场景**：
*   **文件操作**：
    > **用户**："帮我读取 `config.json` 文件的内容。"
    >
    > **AI**：(自动调用 `read_file` 工具) "这是文件的内容：..."
*   **系统控制**：
    > **用户**："当前目录下有哪些大文件？"
    >
    > **AI**：(自动调用 `execute_command` 运行 `du -h`) "发现以下大文件..."

## ⚠️ 注意事项
*   **API 配置**：使用前请确保已正确配置 API Key。
*   **隐私安全**：AI 的回答和工具调用依赖于您配置的模型，建议使用 DeepSeek 等高智商模型以获得最佳体验。
