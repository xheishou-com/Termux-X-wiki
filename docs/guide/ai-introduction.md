# 🤖 Termux-X AI 助手概览

Termux-X 不仅仅是一个终端模拟器，它深度集成了先进的大语言模型（LLM），致力于打造移动端最强大的 AI 辅助开发与渗透平台。通过内置的 AI 引擎，您可以体验从简单的命令生成到复杂的全自动代码构建等一系列革命性功能。

## 🌟 核心能力矩阵

Termux-X 的 AI 生态包含以下五大核心模块：

### 1. [AI 配置](/guide/ai-configuration)
一切智能化的基础。支持自定义 API Key、模型端点（OpenAI/DeepSeek 等）以及 MCP（Model Context Protocol）工具链的配置。

### 2. [命令辅助](/guide/command-assistant)
**"说人话，行命令。"**
无需死记硬背复杂的 Linux 参数。只需输入“压缩当前目录所有图片”或“扫描局域网主机”，AI 即刻为您生成精准的 Shell 命令。

### 3. [AI 终端](/guide/ai-terminal)
**"终端里的贴身专家。"**
集成了 Gemini CLI 和 iFlow 工作流引擎。在命令行中直接与 AI 对话，支持管道操作与系统级交互，是渗透测试与系统管理的得力助手。

### 4. [AI 对话](/guide/ai-code-audit)
**"不仅是聊天，更是控制台。"**
图形化的对话界面，支持多轮上下文记忆。通过 MCP 协议，AI 可以直接读取文件、执行脚本、分析报错，甚至帮您整理文件系统。

### 5. [AI IDE](/guide/ai-code-generation)
**"移动端的 Copilot。"**
全功能的代码编辑器。开启 **Builder Mode** 后，AI 可以从零开始为您编写整个项目文件（如 Python 爬虫、Web 页面），并支持一键运行与调试。

---

## 🚀 为什么选择 Termux-X AI？

*   **本地化体验**：深度集成 Termux 环境，AI 生成的命令可直接执行。
*   **工具链整合**：通过 MCP 协议，打通了 AI 与操作系统之间的壁垒。
*   **多模型支持**：兼容 OpenAI 接口规范，支持接入 DeepSeek、Claude 等各种强大的模型。
*   **移动生产力**：让手机瞬间变身为高效的生产力工具，随时随地编写代码或进行安全测试。
