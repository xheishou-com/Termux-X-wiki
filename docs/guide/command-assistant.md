# 🤖 命令辅助 (Command Helper)

Termux-X 内置了智能命令辅助功能，旨在帮助用户将自然语言需求自动转化为可执行的 Linux/Termux 命令。无论您是记不住复杂参数的新手，还是追求效率的资深玩家，这个功能都能显著提升您的操作体验。

## 🌟 核心功能

*   **自然语言转译**：直接用中文描述你想做的事（例如：“查找大于100M的文件”），AI 会自动生成对应的 Shell 命令。
*   **多环境执行**：
    *   **Run Termux**: 在当前 Termux 普通环境中执行命令。
    *   **Run Kali**: 将命令发送到 Kali NetHunter (Root) 环境中执行（需已安装 NetHunter）。
*   **安全可控**：生成的命令在执行前会展示给用户确认，避免 AI 幻觉导致的误操作。

## 🚀 使用教程

### 1. 开启命令助手
1.  在 Termux-X 主界面，向右滑动打开**侧边栏**。
2.  在侧边栏的功能列表中，找到并点击 **“命令辅助”** 图标（通常是一个机器人头像）。

### 2. 生成命令
1.  在弹出的对话框输入框中，用自然语言描述您的需求。
    *   *示例 1*: `扫描局域网内所有在线的主机`
    *   *示例 2*: `压缩当前目录下所有 jpg 图片为 archive.zip`
    *   *示例 3*: `用 nmap 扫描 192.168.1.1 的开放端口`
2.  点击 **“生成 (Generate)”** 按钮。
3.  稍等片刻，下方输出框将显示 AI 生成的精准命令。

 ![](/img/AI/21.jpg)

### 3. 执行命令
确认生成的命令无误后，根据您的需求选择执行环境：
*   **点击 "Run Termux"**: 命令将立即插入到您当前的 Termux 终端并执行。
*   **点击 "Run Kali"**: 命令将发送到后台运行的 Kali NetHunter 容器中执行（适合需要 Root 权限的渗透测试工具）。

## 💡 使用技巧与示例

### 场景一：文件管理
> **用户输入**: “删除当前目录下所有 .log 后缀的文件”
>
> **AI 生成**: `find . -name "*.log" -type f -delete`

### 场景二：系统监控
> **用户输入**: “查看占用 CPU 最高的 5 个进程”
>
> **AI 生成**: `ps -eo pid,ppid,cmd,%mem,%cpu --sort=-%cpu | head -n 6`

### 场景三：网络工具 (Kali)
> **用户输入**: “使用 sqlmap 检测 http://example.com/id=1 是否有注入漏洞”
>
> **AI 生成**: `sqlmap -u "http://example.com/id=1" --batch`
>
> *(此时建议点击 **Run Kali** 以调用 Kali 中完整的工具链)*

---

## ⚠️ 注意事项

1.  **AI 配置依赖**：使用此功能前，请确保您已在 [AI 配置](/guide/ai-configuration) 中正确设置了 API Key 和模型（推荐使用 DeepSeek 或 GPT-3.5/4）。
2.  **准确性检查**：虽然 AI 生成的命令通常很准确，但在执行涉及删除文件 (`rm`) 或系统修改的危险命令前，请务必人工核对一遍。
3.  **Root 权限**：如果您选择 "Run Kali"，请确保您的设备已获取 Root 权限并正确安装了 Kali NetHunter 环境。
