# 🚀 Termux-X 入门指南概览

欢迎开始您的 Termux-X 之旅！本模块旨在帮助您从零开始，快速掌握 Termux-X 的安装、配置及核心使用技巧。无论您是 Linux 新手还是资深极客，这里都有您需要的内容。

## 📚 模块内容

### 1. [快速开始](/guide/getting-started)
**"一分钟上手。"**
包含应用下载、首次启动引导以及基础环境的初始化设置。这是您体验 Termux-X 的第一步。

### 2. [安装教程](/guide/installation)
**"详细的图文步骤。"**
针对不同 Android 版本（特别是 Android 10+）的详细安装指引，确保您能够顺利完成部署。

### 3. [故障排除](/guide/first-install-error)
**"遇到问题先看这里。"**
汇总了首次启动时可能遇到的常见报错（如 Process 255、权限不足、闪退等）及其解决方案。

### 4. [使用技巧](/guide/usage-tips)
**"像极客一样使用。"**
掌握高效的手势操作、快捷键配置、字体缩放以及侧边栏的高级用法，让您的操作效率翻倍。

---

## 💡 建议学习路径

1.  先阅读 **[快速开始](/guide/getting-started)** 完成安装。
2.  如果遇到安装问题，查阅 **[故障排除](/guide/first-install-error)**。
3.  成功运行后，浏览 **[使用技巧](/guide/usage-tips)** 熟悉操作。
4.  最后，您可以前往 **AI 助手** 或 **Kali NetHunter** 模块探索更多高级功能。
