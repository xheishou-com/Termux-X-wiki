# Termux-X 最新版本安装教程

本教程将指导您如何在 Android 设备上安装最新版本的 Termux-X。

## 1. 环境要求

* **设备架构:** ARM64 (aarch64)
* **Android 版本:** Android 7.1 或更高版本

## 2. 安装步骤

### 步骤一：彻底卸载Termux

在安装 Termux-X 之前，**必须** 彻底卸载您设备上的Termux应用程序。这包括：

* Termux (主程序)

**请务必检查您的应用程序列表，确保所有相关组件均已完全卸载，否则可能会导致安装失败或运行冲突。**

* Termux (组件程序)

![](/img/Termux-X/10.jpg)


### 步骤二：下载最新版 APK

1.  打开您手机上的任意浏览器（如 Chrome, Firefox 等）。
2.  访问 Termux-X 官方最新版获取地址：[termux-x.com](https://termux-x.com/)
3.  浏览器将自动开始下载最新版本的 `.apk` 安装包。

### 下载最新版 APK 注意事项

* 请确保您的设备已连接到互联网。
* 下载速度可能会受到您的网络环境影响。


如果您在下载过程中遇到如下图所示的“应用涉及非法内容”、“高风险应用”等提示，请**无需惊慌**。

![](/img/Termux-X/28.png)

这种情况通常是因为手机系统的安全防护机制（如手机管家、安装器检测）将应用**误判为风险软件**。

**原因分析：**
1.  **高风险权限**：Termux-X 需要 `REQUEST_INSTALL_PACKAGES`（安装应用）和 `MANAGE_EXTERNAL_STORAGE`（所有文件访问）等敏感权限才能正常工作，这些权限通常被安全软件重点监控。
2.  **敏感关键词与功能**：应用集成了 "Kali", "NetHunter", "Exploit", "Root" 等渗透测试相关功能和资源，符合安全软件的“黑客工具”特征库。
3.  **应用签名**：目前使用的签名文件未经过各大手机厂商（华为、小米、OPPO等）的官方白名单认证。

**解决方案：**
1.  **忽略风险安装**：在安装界面点击“更多详情”或“了解风险”，选择**“无视风险安装”**或**“继续安装”**。
2.  **断网安装**：尝试断开网络（关闭 WiFi 和流量）后再进行安装，有时可以绕过云端检测。
3.  **关闭安全检测**：暂时关闭手机管家中的“应用安全检测”功能。

**结论**：这是一个**误报**。应用的性质（终端模拟器/渗透工具）决定了它会被视为潜在风险，并不代表应用本身包含病毒。


### 步骤三：安装 Termux-X

1.  下载完成后，打开您手机的“文件管理器”应用，找到刚刚下载的 `.apk` 文件（通常在 "Download" 或 "下载" 文件夹中）。
2.  点击该 `.apk` 文件。
3.  系统可能会提示您需要授权“允许来自此来源的应用安装”。请根据提示前往设置并开启该权限。
4.  返回安装界面，点击“安装”按钮。
5.  等待安装完成即可。

## 3. 注意事项

* `https://termux-x.com/` 是获取最新版本的唯一官方推荐链接，该链接始终指向最新的稳定版本。
* 重复强调：**必须** 卸载 Termux ，这是确保 Termux-X 正常  安装的最重要前提。
* 安装完成后请务必查看下一个教程以完成首次启动 Termux-X。
* **安装失败（应用未安装/签名冲突）：**

  如果您在安装时提示“应用未安装”或“安装失败”，通常是因为设备上仍残留有旧版本的 Termux 或其插件（如 Termux:API, Termux:Styling, Termux:Widget 等）。
  
  **解决方法：** 请仔细检查手机的应用列表，**卸载所有** Termux 开头的相关应用，然后再次尝试安装。

  ![](/img/Termux-X/11.jpg)

 

* **安装出现风险提示说明：**

  由于 Termux-X 集成了众多网络安全工具和底层系统调用功能，部分手机厂商的“安全管家”或 Google Play 保护机制可能会将其误报为“风险软件”或“病毒”。
  
  **请放心，这是正常现象。** Termux-X 旨在为专业用户提供渗透测试环境，本身不包含任何恶意扣费或窃取隐私的代码。
  
  **解决方法：**
  1. 点击提示框中的 **“更多详情”** 箭头。
  2. 选择 **“仍要安装”** 或 **“单次允许”**。

  ![](/img/Termux-X/12.jpg)
  
* **解析错误（解析软件包时出现问题）：**

  如果您在打开 APK 文件时提示“解析错误”，通常是因为：
  1. **下载文件损坏**：网络不稳定导致下载不完整。请重新下载。
  2. **系统版本过低**：Termux-X 需要 Android 7.0+。如果您的系统低于此版本，将无法安装。
