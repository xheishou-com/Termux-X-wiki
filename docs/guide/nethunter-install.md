# 📥 安装/重装 Kali NetHunter

在 Termux-X 中安装 Kali NetHunter 非常简单，我们为您提供了全自动化的安装脚本。

## 🚀 安装步骤

1.  打开 Termux-X，向右滑动打开**侧边栏**。
2.  在 **Kali NetHunter** 区域，找到并点击 **"安装/重装"** 图标（通常是一个下载箭头）。
3.  系统会自动弹出一个确认对话框，询问是否开始安装。
4.  点击 **"确认"** 后，终端将自动执行下载和解压流程。
    *   *注意：安装包较大（约 200MB - 1GB），请确保在 WiFi 环境下进行，并保持网络通畅。*
5.  等待终端提示 `Installation successful` 或类似完成信息。

## 🔄 重装系统

如果您的 Kali 环境出现无法修复的错误，或者您想重置到初始状态：

1.  重复上述安装步骤。
2.  在弹出的对话框中选择 **"重装"** 或 **"覆盖"**。
3.  **警告**：重装操作会**清除**您在 Kali 环境中安装的所有工具和保存的文件，请务必提前备份重要数据！

## ⚠️ 常见问题

*   **下载速度慢/失败**：
    *   请检查网络连接。
    *   尝试在设置中切换下载源（如果有该选项）。
*   **解压失败**：
    *   通常是因为下载文件损坏。请尝试删除下载的镜像文件后重试，或直接点击重装。
    *   确保存储空间充足（建议预留 5GB 以上）。
