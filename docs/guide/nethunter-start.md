# 💻 启动 Kali NetHunter 终端

安装完成后，您就可以随时启动 Kali 终端，体验强大的渗透测试工具了。

## 🚀 启动方法

1.  打开 Termux-X，向右滑动打开**侧边栏**。
2.  在 **Kali NetHunter** 区域，点击 **"启动终端"** 图标（通常是一个控制台符号 `>_`）。
3.  界面将直接跳转到一个新的终端会话，提示符通常会变为 `root@localhost` 或 `kali@localhost`。

## 📝 终端环境说明

进入 Kali 终端后，您将处于一个独立的 Linux 环境中：

*   **用户权限**：
    *   默认通常以 `root` 权限进入（在容器内），这意味着您可以随意安装软件和修改系统文件。
    *   *注意：这里的 Root 是容器内的 Root，并不等同于安卓宿主机的 Root 权限（除非您使用的是 Root 模式）。*
*   **包管理器**：
    *   使用 `apt` 命令来管理软件。
    *   例如：更新源 `apt update`，安装 git `apt install git`。
    *   *切记：不要使用 `pkg` 命令，那是 Termux 原生环境的包管理器。*
*   **文件系统**：
    *   您的安卓文件系统通常挂载在 `/sdcard` 目录下，方便您在 Kali 和安卓之间传输文件。

## 🚪 退出终端

完成操作后，建议正确退出以节省资源：

*   在命令行输入 `exit` 并回车。
*   或者直接关闭该终端会话窗口。
