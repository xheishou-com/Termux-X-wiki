# ☁️ Termux-X 下载站

**下载站** 是 Termux-X 的资源中心，提供各类 APP 插件、系统镜像、常用工具包的高速下载。所有资源均托管于官方高速服务器，支持断点续传。

## 🚀 进入方式

1.  **打开侧边栏**：从屏幕左侧向右滑动。
2.  **点击图标**：找到 **"下载站"** 按钮（图标为云端下载样式）。
3.  **连接服务器**：系统会自动连接到官方文件服务器并加载资源列表。

## 📦 资源类别

下载站主要提供以下类型的资源：

### 📱 常用插件 (APKs)
*   **Termux:API**：提供硬件调用接口（如相机、传感器）。
*   **Termux:Styling**：终端配色与字体美化工具。
*   **Termux:Widget**：在桌面创建脚本快捷方式。
*   **Termux:Boot**：开机自启动脚本管理。
*   **VNC Viewer**：推荐的图形化桌面连接客户端。

### 🖥️ 系统镜像
*   **Windows 镜像**：用于 QEMU 虚拟机的 WinXP/Win7 精简版镜像。
*   **Linux 根文件系统**：各类 Linux 发行版的 Rootfs 包。

### 🛠️ 辅助工具
*   **黑客工具包**：部分较大的渗透测试工具包（如 Metasploit 离线包）。
*   **字体包**：编程专用字体集合。

## ✨ 功能特性

*   **⚡ 高速下载**：接入 CDN 加速，解决 GitHub 等源站下载慢的问题。
*   **⏯️ 断点续传**：支持暂停和恢复下载，网络波动也不怕。
*   **🤖 自动安装**：下载 APK 文件后，支持一键调用系统安装程序。
*   **📂 智能路径**：
    *   **APK**：自动保存至 `Termux-X/Apk` 目录。
    *   **数据文件**：自动保存至 `Termux-X/Data` 目录。

## ⚠️ 常见问题

**Q: 提示“无法连接到下载站服务器”？**
A: 请检查您的网络连接。如果服务器维护中，请稍后重试或关注社区公告。

**Q: 下载速度慢？**
A: 下载速度取决于您的网络环境。如果您使用移动数据，请注意流量消耗。

**Q: 下载的 APK 无法安装？**
A: 请确保您已授予 Termux-X "安装未知应用" 的权限。
