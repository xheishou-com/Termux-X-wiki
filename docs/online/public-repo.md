# 🔓 公共仓库 (自定义仓库连接)

**公共仓库** (Public Repository) 是 Termux-X 提供的一项允许用户连接到第三方或私有资源服务器的功能。虽然名为“公共仓库”，但它本质上是一个 **自定义仓库连接器**。

## 📝 功能说明

通过此功能，您可以输入一个 HTTP 服务器地址，Termux-X 将尝试连接该地址以获取资源列表。这对于企业内部部署或通过局域网共享脚本和工具非常有用。

## 🚀 如何使用

1.  在 Termux-X 侧边栏菜单中找到 **公共仓库** 选项。
2.  点击后会弹出一个输入框。
3.  输入目标仓库服务器的地址 (URL)。
    *   默认显示的地址 `http://10.242.164.19` 通常是一个示例或内网测试地址，**不可直接使用**。
    *   请输入您自己搭建的或已知的有效仓库地址。
4.  点击 **确定**，App 将尝试连接并加载资源。

## ⚠️ 注意事项

*   **安全风险**：请仅连接您信任的服务器地址。连接恶意服务器可能会导致恶意脚本被下载或执行。
*   **服务器搭建**：如果您想搭建自己的仓库，通常需要运行一个简单的 HTTP 服务，并按照 Termux-X 支持的 JSON 格式 (`repository/main.json`) 提供资源列表。
